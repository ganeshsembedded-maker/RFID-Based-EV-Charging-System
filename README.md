# RFID-Based-EV-Charging-System
RFID-based EV charging system using ATmega328 for secure user authentication and automated charging control.
## Project Description
This project demonstrates an **RFID-based electric vehicle (EV) charging system** using a microcontroller. The system allows **authorized users** to charge their EVs securely using RFID cards.

This repository contains the **project report**, which includes:  
- Project objectives  
- Circuit diagram  
- Working principle  
- Components used  
- Future scope
